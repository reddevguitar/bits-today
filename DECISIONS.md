# DECISIONS

- Project title: `bit's today`
- Version 1 is a Bitcoin monitoring and decision-support agent
- The agent runs every 2 hours
- It checks BTC status, relevant news, chart context, and notable public-figure investment news when fresh and relevant
- It outputs a human-readable judgment and updates a paper-trading portfolio automatically
- Initial virtual capital is 10,000,000 KRW
- Real exchange execution remains out of scope for now
